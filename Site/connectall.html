Connect All

Insurance Agent Ticket Software
x
  • Categories: Business
  • Released: 15 December 2016 - Ongoing
  • Online: www.caapp.co
  • Client: Kevin Trinh
Description

Connect All is an insurance agent ticketing software that allows insurance agents the ability to create tickets for their individual clients. The software allows you to manage different policies, determine the notes and the user that created the note and give administrative view for the manager of the insurance group. Twilio integration will allow customers to call in and their profile display immediately for the agent. The site is built on React, Redux and Node.
